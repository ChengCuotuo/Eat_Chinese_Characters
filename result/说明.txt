小游戏，编写环境
Java(TM) SE Runtime Environment (build 1.8.0_181-b13)
Java HotSpot(TM) 64-Bit Server VM (build 25.181-b13, mixed mode)
需要 jre 运行环境，自行下载

使用W,A,S,D控制方向
两种模式，经典模式和和汉字模式
经典模式，在每次暂停重新开始之后，作为惩罚会多出两个点
汉字模式，根据文本框中输入的汉字，在点击确认后，顺次数显汉字点阵，作为食物

两种模式，只有在红色的开始按钮变成“Pause”或“Finish”的时候，才可以转换。否则无效

中间的数字组合框，用来选择速度
